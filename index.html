Grafuri Neorientate

Introducere

Grafurile neorientate sunt structuri matematice utilizate pentru a modela relații între obiecte. În această lucrare, vom explora diferite concepte și aplicații ale grafurilor neorientate.

Teorie

Ce sunt grafurile neorientate?

Un graf neorientat este o structură de date formată dintr-o mulțime de noduri (sau vârfuri) și o mulțime de arce care le leagă. Spre deosebire de grafurile orientate, arcele nu au o direcție specifică.

Proprietăți

  • Conectivitate
  • Cicluri
  • Arbori și componente conexe

Algoritmi importanți

  • DFS (Depth First Search)
  • BFS (Breadth First Search)
  • Algoritmul Dijkstra pentru cel mai scurt drum

Implementare

BFS (Breadth First Search)


function bfs(graph, startNode) {
    let visited = new Set();
    let queue = [startNode];
    while (queue.length > 0) {
        let node = queue.shift();
        if (!visited.has(node)) {
            visited.add(node);
            console.log(node);
            let neighbors = graph[node] || [];
            for (let neighbor of neighbors) {
                queue.push(neighbor);
            }
        }
    }
}
let graph = {A: ['B', 'C'], B: ['A', 'D'], C: ['A', 'D'], D: ['B', 'C']};
bfs(graph, 'A');

DFS (Depth First Search)


function dfs(graph, startNode, visited = new Set()) {
    if (visited.has(startNode)) return;
    visited.add(startNode);
    console.log(startNode);
    let neighbors = graph[startNode] || [];
    for (let neighbor of neighbors) {
        dfs(graph, neighbor, visited);
    }
}
let graph = {A: ['B', 'C'], B: ['A', 'D'], C: ['A', 'D'], D: ['B', 'C']};
dfs(graph, 'A');

Dijkstra


function dijkstra(graph, startNode) {
    let distances = {};
    let visited = new Set();
    distances[startNode] = 0;
    let nodes = Object.keys(graph);

    while (nodes.length > 0) {
        let closestNode = nodes.reduce((closest, node) => 
            distances[node] < distances[closest] ? node : closest, nodes[0]);
        
        nodes = nodes.filter(node => node !== closestNode);
        let neighbors = graph[closestNode] || {};
        
        for (let neighbor in neighbors) {
            let distance = distances[closestNode] + neighbors[neighbor];
            if (distance < (distances[neighbor] || Infinity)) {
                distances[neighbor] = distance;
            }
        }
    }

    return distances;
}
let graph = {A: {'B': 5, 'C': 2}, B: {'A': 5, 'C': 1}, C: {'A': 2, 'B': 1}};
console.log(dijkstra(graph, 'A'));

Concluzii

Grafurile neorientate sunt structuri esențiale în informatică și au multiple aplicații în domenii precum rețelele de calculatoare, rețelele sociale și optimizarea traseelor. Aceste structuri sunt folosite în analiza și rezolvarea problemelor de conectivitate și flux de informație. Algoritmii precum BFS, DFS și Dijkstra sunt instrumente fundamentale pentru explorarea grafurilor. BFS este utilizat pentru a explora grafurile pe niveluri, DFS pentru a explora în profunzime, iar Dijkstra pentru a găsi cel mai scurt drum între noduri într-un graf ponderat.
